JavaScript: Linguagem de Programação da Web

JavaScript é a linguagem que adiciona interatividade, lógica e dinamismo às páginas web. Ele é executado no navegador e também pode ser usado no servidor (Node.js).

Funções Principais

  • Manipular o DOM (elementos HTML).
  • Responder a eventos de usuário.
  • Fazer requisições a APIs.
  • Executar lógica de programação completa.

Exemplo Básico


alert("Olá, mundo!");
document.querySelector("h1").style.color = "blue";

Sintaxe Básica do JavaScript

O JavaScript usa variáveis, operadores, condicionais e loops semelhantes a outras linguagens.

Variáveis


let nome = "Pedro";
const idade = 20;
var ativo = true;

Condicionais


if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}

Loops


for (let i = 0; i < 5; i++) {
    console.log(i);
}

Funções e Escopo

Funções permitem reutilizar código e organizar lógica.

Declaração e chamada


function somar(a, b) {
    return a + b;
}
console.log(somar(3, 4));

Funções de seta (arrow functions)


const dobro = x => x * 2;
console.log(dobro(5));

Objetos e Arrays

Objetos armazenam dados em pares chave/valor e arrays armazenam listas ordenadas.

Objetos


const pessoa = {
    nome: "Ana",
    idade: 25,
    saudacao() {
        console.log("Olá, " + this.nome);
    }
};
pessoa.saudacao();

Arrays


const frutas = ["maçã", "banana", "uva"];
frutas.forEach(f => console.log(f));

Manipulação do DOM

O DOM (Document Object Model) representa a estrutura do HTML como objetos manipuláveis via JS.


const titulo = document.querySelector("h1");
titulo.textContent = "Título alterado por JavaScript";
titulo.style.color = "red";

Eventos e Interatividade

Eventos permitem responder a ações do usuário (cliques, teclas, rolagem, etc.).


const botao = document.querySelector("button");
botao.addEventListener("click", () => {
    alert("Botão clicado!");
});

Assincronismo: Promises e Async/Await

O JavaScript executa tarefas assíncronas para evitar travamentos e melhorar desempenho.

Promises


fetch("https://api.exemplo.com/dados")
    .then(res => res.json())
    .then(data => console.log(data))
    .catch(err => console.error(err));

Async/Await


async function carregarDados() {
    try {
        const res = await fetch("https://api.exemplo.com/dados");
        const data = await res.json();
        console.log(data);
    } catch (erro) {
        console.error(erro);
    }
}
carregarDados();

Boas Práticas de JavaScript

Escreva código limpo, seguro e eficiente.

  • Use const e let em vez de var.
  • Prefira funções pequenas e reutilizáveis.
  • Evite poluir o escopo global.
  • Trate erros com try/catch.
  • Use === em vez de ==.
  • Documente funções e mantenha estilo consistente.
